# Week 1 — AWS Account Foundation & Security Baseline

This guide walks through securing and preparing your AWS environment for a startup.

## Scope
- Root account hardening
- IAM Identity Center (SSO)
- Cost controls and alerts
- CloudTrail and logging
- AWS Organizations


## Outcome
By the end of this week:
- Your AWS account is secure
- All access is federated (no root usage)
- Costs are monitored
- All API activity is logged
- You are ready for multi-account setup (Week 2)

---

## Structure

- Day 1-2 → Root Security, IAM Identity Center
- Day 3 → Cost Controls
- Day 4 → CloudTrail & Logging
- Day 5 → AWS Organizations
